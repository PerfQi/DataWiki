# DataWiki

## 1.数据中台是什么？
https://zhuanlan.zhihu.com/p/68584597
数据中台就是企业的数据仓库+数据服务工厂，完成从数据到价值的加工，服务于业务、决策等。

数据中台由三个部分组成： 数据服务中心、服务管理工具、运营管理机制。

其中最核心的是数据服务中心，包含垂直数据汇聚区、公共数据服务区、萃取数据服务区以及高级数据服务区。

服务管理工具、运营管理机制主要维护数据中台的“稳定”。
